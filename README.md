#JOKE GENERATION PROJECT

## Description
This is a javascript project that generates random jokes using the joke API. when the user clicks a button, a joke is fetched and it is displayed on the page

## Features
- Shows a loading joke text
- Generates a random joke when button is clicked
- Handles both single-line and double line jokes
- Shows the joke on the browser

Getting Started
1. Clone the repository
```bash git clone https://github.com/MoyosoreCoder/jokegeneratorAi.git
```
2. Open the project folder
3. Open index.html in your browser or open live server option

API used
https://v2.jokeapi.dev/joke/Any
